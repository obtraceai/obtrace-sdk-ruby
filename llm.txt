Obtrace Ruby SDK — add observability to your backend.

Obtrace is a hosted SaaS (like Datadog or Sentry). This is a client library.
Install it, add one init call, done. DO NOT create wrapper files or config modules.

Install:
  gem "obtrace-sdk-ruby"

Setup (add to your app entry point):
  require "obtrace_sdk"
  client = ObtraceSDK::Client.new(ObtraceSDK::Config.new(
    api_key: ENV["OBTRACE_API_KEY"],
    service_name: ENV["OBTRACE_SERVICE_NAME"],
  ))

What it captures automatically:
- Rack/Rails HTTP requests and outbound HTTP calls
- Logs, errors, and performance traces

Environment variables:
  OBTRACE_API_KEY       — your project key (from app.obtrace.ai)
  OBTRACE_SERVICE_NAME  — name of this service (e.g. api-service)
  OBTRACE_ENVIRONMENT   — deployment environment (e.g. production, staging)

Docs: https://docs.obtrace.ai/sdks/ruby
